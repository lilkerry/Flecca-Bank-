# Flecca Bank Robbery System

**Realistic Bank Robbery Script for FiveM - QBCore/QBX**

## Features
            KERRYGAMER
✅ **3 Bank Locations** - Del Perro, Downtown, Pillbox  
✅ **Realistic Robbery Mechanics** - Hacking, drilling, stealth options  
✅ **Police Response System** - Real-time dispatch notifications  
✅ **Dynamic Loot System** - Random items, cash, diamonds, gold  
✅ **Crew System** - Multiple players can rob together  
✅ **Alarm System** - Silent/loud alarm with police dispatch  
✅ **Cooldown System** - Banks go on lockdown after robbery  
✅ **Server Logging** - All robberies tracked and logged  
✅ **Fully Customizable** - All settings in config.lua  
✅ **Export Functions** - Integration with other scripts
