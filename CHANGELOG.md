# Changelog - Flecca Bank Robbery System

## [1.0.0] - 2026-05-19 
             KERRYGAMER
### Added
- Initial release of Flecca Bank Robbery System
- 3 bank locations (Del Perro, Downtown, Pillbox)
- Realistic robbery mechanics
- Hacking and drilling systems
- Police dispatch and response
- Dynamic loot system
- Crew management system
- Server-side logging
- Alarm system with notifications
- Bank cooldown system
- Full configuration options
- Export functions for integration
- Comprehensive documentation

### Features
- Multi-location support
- Configurable police requirements
- Reward system with crew sharing
- Item spawning and distribution
- Real-time notifications
- Server logging to file
- Debug mode for testing

### Scripts Included
- fxmanifest.lua - Resource manifest
- shared/config.lua - Configuration
- client/client.lua - Main client logic
- client/effects.lua - Visual/audio effects
- client/utils.lua - Utility functions
- server/server.lua - Main server logic
- server/logs.lua - Logging system
